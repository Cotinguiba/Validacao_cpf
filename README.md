# Validador de CPF em Java

## 📌 Sobre o Projeto

Este projeto é um programa simples em Java que realiza a validação de um CPF brasileiro utilizando o cálculo oficial dos dígitos verificadores.

O sistema:

- Recebe um CPF digitado pelo usuário;
- Remove caracteres especiais automaticamente;
- Verifica se o CPF possui 11 dígitos válidos;
- Calcula os dois dígitos verificadores;
- Informa se o CPF é válido ou inválido.

---

## 🚀 Funcionalidades

✅ Entrada de CPF pelo usuário  
✅ Remoção automática de pontos e traços  
✅ Verificação de tamanho do CPF  
✅ Bloqueio de CPFs com números repetidos  
✅ Cálculo do primeiro dígito verificador  
✅ Cálculo do segundo dígito verificador  
✅ Validação final do CPF  

---

## 🛠️ Tecnologias Utilizadas

- Java

---

## 📂 Estrutura do Código

O programa é dividido em etapas:

1. Entrada do CPF
2. Tratamento dos dados
3. Verificação inicial
4. Cálculo do primeiro dígito
5. Cálculo do segundo dígito
6. Comparação dos dígitos verificadores
7. Exibição do resultado
